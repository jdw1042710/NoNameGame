<h2> 아무게임이나 만들어보기 프로젝트 </h2>
평소에 공부했거나 공부하고 싶었던 기능을 한데 모아 게임으로 만드는 프로젝트입니다.<br>
3인칭 시점의 어드벤쳐 게임을 베이스로 하였고, 다양한 게임적 요소들을 구현하여 집어넣고 있습니다.
</div>

## 개요
> - 프로젝트 기간: 2024.02 ~ 진행중
> - 개발 엔진: Unity (2022.3.13f1 version)
